Welcome to my portfolio!
I am Ignacio Manuel Castro Game designer and developer
I've recently finished my game design and development studies at "Escuela Da vinci de arte multimedia".
These are some of the projects i have worked on during the last few years

Astrobotanic

Astrobotanic is a 3D isometric view dungeon crawler game in which you control an astronaut whose arm has become a mutant plant.
Using your new arm you'll have to face enemies solve puzzles and find out how to cure yourself and get back your arm.
This project is stil under development as my thesis project to get my degree at "Escuela Da vinci"
I'm working as game designer and programmer together with other 4 friends using Unity

  • Latest build

Shadow Lords

Shadow lords is a roleplaying and management game in which you'll take the role of a feudal lord trying to manage a small town as it grows.
Every in-game day new problems and events will happen around the town forcing you to act as a mediator and solve your people's issues
I was the main game designer and programmer in this project in which we worked 4 months
It was made using Unity and Inkle API to manage a complex dialogue tree system and to allow for the addition of new events and dialogue without having to write any code

Ashes of Eric Rivel

Ashes of Eric Rivel is a first person horror game
In this game the player will enter a haunted victorian university to find out what happened to his dear friend Eric
Here i also worked as a game designer (mostly working on puzzles) and programmer
Our main goal with this project was to learn how to work using the HDRP render pipeline in Unity

  • Latest build

Los pinturas

Los pinturas is a small multiplayer minigame in which two players compete trying to paint more parts of the arena than their adversary
It's a really small project which i made with two friends to learn the basics of how to use Photon fusion in less than a week
This time i worked mostly as a programmer

  • Latest build

Unreal Magic

Unreal Magic is a First person shooter game in which the player can control metallic objects to mentally push or pull them or himself
Me and two other friends made this project with two goals in mind, learning how to use Unreal Engine blueprints and making a prototype of how the powers described in the Mistborn book series by Brandon Sanderson would work in a videogame.
I worked both as a programmer and as a game designer in this project.
Almost all assets are placeholders because we wanted to focus on learning the engine rather than making something pretty

  • Latest build

Orbital Cleanup

Orbital Cleanup is a 3rd person mobile game in which the player takes up the role of an astronaut who's tasked with repairing a satellite and collecting trash around it
This is another project i made with friends which we used to learn how to make games for Android devices using Unity.
As always i worked as both a programmer and game designer for the project.
